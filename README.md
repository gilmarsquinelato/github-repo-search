# Github repository search

This repository is an exercise on how to implement the Github search functionaliry using its GraphQL endpoint.

## Stack
* React, using create-react-app
* Typescript
* Apollo GraphQL
* Material UI
* Emotion
* Jest
* Cypress
* ESLint + Prettier
* Husky

## Running locally
```bash
yarn start
```

## Running on production
```bash
yarn build
npx serve -s ./build
```
